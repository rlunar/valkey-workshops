# Module 1: Why Caching is Important?

## Overview

In this module, you'll explore the fundamental importance of caching and see its benefits demonstrated through a real-world airport application.

## Learning Objectives

- Understand the performance benefits of caching
- Explore the sample Airport App and visualize caching improvements
- Learn the AirportDB (FlughafenDB) schema
- Use mycli to run sample queries and validate latency
- Understand query plans with the EXPLAIN command
